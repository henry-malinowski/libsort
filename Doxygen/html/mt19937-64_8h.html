Detailed Description

A C-program for MT19937-64 (2004-09-29 version). Coded by Takuji Nishimura and Makoto Matsumoto.

This is a 64-bit version of Mersenne Twister pseudorandom number generator.

Before using, initialize the state by using mt_seed(seed) or init_by_array64(init_key, key_length).

References:

[1] M. Matsumoto and T. Nishimura, ``Mersenne Twister: a 623-dimensionally equidistributed uniform pseudorandom number generator''

[2] T. Nishimura, ``Tables of 64-bit Mersenne Twisters'' ACM Transactions on Modeling and Computer Simulation 10. (2000) 348–357.

[3] ACM Transactions on Modeling and Computer Simulation 8. (Jan. 1998) 3–30.

Any feedback is very welcome. http://www.math.hiroshima-u.ac.jp/~m-mat/MT/emt.html
