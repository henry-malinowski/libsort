int issorted ( const void *  base,
size_t  nmbers,
size_t  size,
int(*)(const void *, const void *)  cmp 
)

Verifies that an array of an arbitrary type is sorted.

This function uses a tight loop to compare each element with the previous element starting from the second element. If the comparison loop breaks early the comparison between iterator and nmbers will fail. Likewise, if the loop is broken be reaching the limit of the array then the length comparison will succeed and the array has been found to be sorted in accordance with cmp.

Parameters
baseA pointer to the beginning of the array.
nmbersThe number of items in the array.
sizeThe size of the individual elements of the array.
cmpA comparator function for checking the "sortedness". This function returns a value less or equal to 0 when array[0] is "less than or equal" to array[1].
Returns
An int acting as a boolean value.
Return values
0if the array is not sorted in accordance with cmp.
1if the array is sorted in accordance with cmp.

Example usage:

issorted(array, sizeof(array)/sizeof(array[0]), sizeof(array[0]), cmp_func);
